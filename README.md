KeenKeeper — Keep Your Friendships Alive
KeenKeeper is a modern friendship management web application that helps you stay connected with the people who matter most. Track your interactions, set contact goals, and never let important relationships fade away.

Technologies Used
Technology	Purpose
React.js (Vite)	Building the user interface
React Router DOM	Client-side page navigation
Tailwind CSS	Styling and responsive design
Recharts	Pie chart for friendship analytics
React Hot Toast	Toast notifications
React Icons	Icon library for UI elements
Key Features
Friend Dashboard — View all your friends at a glance with status-coded cards showing who's overdue, almost due, or on track for contact.

Quick Check-In System — Log calls, texts, and video interactions with a single click. Each check-in is automatically recorded to your timeline with toast notifications.

Friendship Analytics — Visualize your interaction patterns with an interactive pie chart showing the breakdown of calls, texts, and video chats across all your friendships.
